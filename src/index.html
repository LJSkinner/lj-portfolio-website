My time at KMSoft

During the summer semester of my 3rd year, I undertook my work placement with KMSoft. They create software and products related to inventory management for a range of different industries, including warehouses and medical. KMSoft is located in codebase in Stirling.

As part of this placement, I was able to develop real world skills by helping the development team with their software products. The first project involved creating a CSV parser using C# and .NET 5.0 that would take in CSV files provided by KMSoft's suppliers and parse out the products. This would then update their database with the new stock and product information. I also helped develop a new database schema for their product infrastructure.

The second project I was involved in was helping to create a MVC front-end, again using C# and .NET 5.0 using the Entity Framework which would allow an administrator to view and manage the products. This would also allow them to keep track of the stock level of the products.

Final Dissertation Project

My final dissertation involved using Machine learning to help guide Genetic Improvement of Software. GI involves applying small granular edits to source code using evolutionary algorithms. For this project I used python and the sk-learn library to train a model to help target these edits by finding regions of source code which are most amenable/editable.
